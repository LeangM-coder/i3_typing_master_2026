# i3_typing_master_2026
# Project Description

I3 Typing Master 2026 is a console-based Java application that allows users to test and improve their typing speed and accuracy. The system provides features such as user registration, login, password reset, typing tests, and result tracking.

# Features
    Welcome Menu Screen
    User Registration
    User Login
    Reset Password
    Typing Test
    View Test Results
    About Us
# Technologies Used
    Java (JDK 21 or higher)
    Console-based interface
    Git & GitHub for version control
# How to Setup & Run
    1. Clone the Repository
    git clone https:https://github.com/LeangM-coder/i3_typing_master_2026
    2. Open Project
    Open the project in your IDE (IntelliJ, Eclipse, VS Code)
    3. Compile the Program
    javac Main.java
    4. Run the Program
    java Main
